# Planogram Builder V2 + PDF (No Admin) — Updated PDF Layout

## Key PDF Behavior (Requested)
- Print/PDF defaults to landscape (CSS @page landscape).
- Left side is a full-height panel showing the segment photo in portrait orientation (FIT / contain).
- Right side is a table grouped by shelf, with product details listed under each shelf.
- If a shelf spans multiple pages, the shelf header repeats with “(cont.)”.

## Run
Open index.html in Chrome/Edge/Safari.

## Print/PDF
Open a segment → "Print / Save as PDF" → Save as PDF in the print dialog.
